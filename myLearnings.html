Learnings

I have learned a lot from working with this project. My previous experiences in coding have been mostly backend focused. This was the first time I coded using JavaScript and HTML. It was also the first time I worked with this big set of uncleaned data. The learning curve has been steep but every day I have experienced progress. Due to time limitations, the result did not turn out as I visualized (ba-dum-tss) it to be.

Even if the project has created sleepless nights I have had a great amount of fun learning D3.js, doing my best to interpret the data, and building this website.
